# Prompt Engineering per Dev

Presentazione reveal.js su tecniche di prompt engineering per sviluppatori.

## Avvio rapido

1. Apri `index.html` nel browser
2. Usa le frecce per navigare tra le slide

## Controlli

- ← → : naviga tra le slide
- ESC : panoramica slide
- S : modalità speaker (note)

## Contenuto

- Tecniche di prompt engineering
- Esempi pratici di refactoring e debug
- GitHub spec-kit workflow
- Best practices

## Dipendenze (CDN)

- Reveal.js 4.5.0
- Highlight.js 11.8.0
- Mermaid 10

Nessuna installazione necessaria - tutto viene caricato via CDN.
